Case Study

Scalable Air Quality Monitoring System for Smart Buildings

This case study details the development and deployment of a scalable air quality monitoring system for a large real estate company. The system uses IoT sensors to collect and analyze air quality data across multiple buildings.

About the project

Client Overview

A major real estate company managing commercial properties across North America.

Problem Statement

The client needed to monitor air quality in their buildings to ensure tenant safety and comply with regulations. They lacked a scalable system to deploy and manage air quality sensors across their property portfolio.

Objectives

  1. Deploy air quality sensors across multiple buildings
  2. Collect and analyze air quality data in real-time
  3. Create a scalable system for easy expansion to new properties
  4. Provide actionable insights for building management
Solution Overview

We developed an end-to-end IoT solution for air quality monitoring. The system includes hardware sensors, a cloud-based data platform, and a web dashboard for data visualization and analysis.

Hardware Solution

We collaborated with the client to implement a cost-effective sensor network for Indoor Air Quality (IAQ) monitoring. The system uses battery-powered "all-in-one" sensors that measure CO2, temperature, and humidity levels. These sensors connect wirelessly over LoRaWAN to on-premise gateways.

For locations without access to the corporate network, we deployed 4G-LTE enabled gateways to ensure connectivity to the cloud-based LoRaWAN server. This setup allows for flexible deployment across diverse office environments. We partnered with a third-party vendor to install sensors using our developed plug-and-play workflow, streamlining the deployment process.

Software Solution

Our software solution centers on a comprehensive Digital Twin platform for managing the full lifecycle of sensor devices. We developed a device management system that handles deployment planning, identity provisioning, configuration, over-the-air updates, and spatial context mapping. This system serves as the backend for a custom iOS mobile app, which simplifies the installation and onboarding process for third-party installers.

We built a robust data pipeline to ingest telemetry data from the sensors. This pipeline maps sensors to their digital twins and enriches raw data with additional context for analysis. The processed data is stored for real-time monitoring and for data warehousing and advanced analytics.

Key Features

Scalable sensor deployment and management.

Automated alerts for air quality issues.

Real-time air quality monitoring.

Historical data analysis and trending.

Integration with existing Building Management Systems.

Conclusion

The air quality monitoring system met all of the client's objectives. We deployed sensors across their property portfolio, enabling real-time air quality data collection and analysis. The scalable system design allows easy expansion to new properties. The web dashboard and analytics engine provide actionable insights for building management. This solution improves tenant safety, ensures regulatory compliance, and optimizes building operations. The client now has a powerful tool for maintaining high air quality standards across their entire real estate portfolio.
