Validating a Unified Global Access Control System Concept for Real Estate Portfolio

This case study examines our work with a global real estate firm to validate a unified Access Control System concept. We developed a prototype to test integration across diverse properties, aiming to improve efficiency and reduce costs. The project provided concrete data to support informed decision-making for potential large-scale implementation.

About the project

Client Overview

An international real estate company managing a diverse portfolio of properties across multiple locations.

Problem Statement

The client faced challenges with inconsistent Access Control Systems (ACS) across their properties. They needed to validate whether a unified, scalable solution could be developed to work with existing systems while enabling seamless, rapid access management across their entire portfolio.

Objectives

  1. Validate the concept of a unified access control platform compatible with various existing systems
  2. Prove the feasibility of real-time, remote access granting and revocation without human operators
  3. Demonstrate potential for reducing installation and ownership costs across the property portfolio
  4. Gather real-world data to support decision-making for full-scale implementation
  5. Develop a working prototype to showcase the potential of the solution
Solution Overview

We developed a prototype Access Control System, including custom hardware and software solutions. The aim was to prove the concept's viability and gather real-world data, and deliver a blueprint for a production-ready system.

Hardware Solution

We engineered a back-room box controller for door relay actuation. This modular hardware supports both Gate-C installations and retrofits. The controller actuates door locks through relays or dry contacts and includes Wiegand I/O for legacy system compatibility. It supports network readers and bridges, enabling flexible integration. Authorization is cloud-provisioned with a local ACL backup, ensuring operation during network outages. The system also allows for remote actuation, enhancing control capabilities.

Software Solution

We developed a Physical Access Control (PAC) system, designed for both on-premises and cloud deployment. This system can provide unified access provisioning across the entire building portfolio, integrating seamlessly with existing building management systems. It delivers real-time analytics and maintains local decision-making capabilities during network outages, while synchronizing with cloud systems when online. This architecture ensures robust, continuous operation regardless of network conditions.

Key Features

Modular hardware compatible with existing Access Control Systems and installed hardware.

Cloud-enabled software with offline capabilities.

Real-time access granting and revocation.

Centralized admin view with multi-system integration.

Analytics capabilities for usage insights.

On-demand, time bounded access provisioning for guests and resourc booking.

Implementation Highlights

Designed and produced a hardware ready for production.

Completed Concept, Prototyping, EVT, and DVT stages of hardware development.

Developed software for user management, analytics, and building system integration.

Created a proof-of-concept unified access provisioning flow.

Implemented a system capable of offline operation.
